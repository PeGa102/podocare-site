# Site do Podocare

Página estática pronta para GitHub Pages. Não requer instalação ou servidor.

## Antes de publicar

1. Quando o novo aplicativo for publicado, abra `app-config.js` e coloque o endereço oficial entre as aspas de `window.PODOCARE_APP_URL`. Todos os botões serão atualizados de uma só vez.
2. Em `privacidade.html` e `termos.html`, substitua todos os textos entre colchetes e `SEU_EMAIL_AQUI`.
3. Solicite revisão jurídica dos documentos legais, pois o Podocare trata dados de saúde.

## Publicação no GitHub Pages

Envie o conteúdo desta pasta para a raiz do repositório (ou para a pasta `docs`). No GitHub, acesse **Settings → Pages**, selecione a branch e a pasta usada, e salve.

O arquivo inicial é `index.html`.
